🔘 MinimapButtonFrame

Collects minimap addon buttons and organizes them into a movable, customizable frame

v3.1.8 WoW 3.3.5a Project Ascension
⬇ Download 📂 View Source

✨ Features

📦 Button Collection

Automatically gathers minimap buttons. Manual add/remove with /mbf add and /mbf remove.

🎨 Customization

Movable and resizable frame, customizable background color, modular button skins.

🔌 ElvUI Integration

Seamless ElvUI toggle - let MBF or ElvUI's Button Grabber control minimap buttons.

🌍 Localization

Supports 9 languages: English, German, French, Spanish, Korean, Russian, Chinese.

🔧 Commands

Command Description
/mbf Display command options
/mbf add Add button under cursor
/mbf remove Remove button from frame
/mbf show Show the MBF frame
/mbf hide Hide the MBF frame
/mbf config Open configuration
